CSS Pseudo Elements - dir

Paragraphs + heading text color set to blue. :ltr sets all text to black if working.
LTR spans set to green. RTL spans set to red.

:dir(ltr) specified

Wrapped with span set to LTR: LTR Text

:dir(ltr) unspecified

Wrapped with span, unset: LTR Text

:dir(rtl) specified

Wrapped with span set to RTL: RTL Text

:dir(ltr) unspecified

Wrapped with span, unset: RTL Text

:dir(rtl) specified - RTL implied

Wrapped with span set to RTL: نص رتل

:dir(ltr) unspecified - RTL implied
